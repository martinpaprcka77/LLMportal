Otázka

Funguje na jakémkoliv modeluBez speciálního kontextu
A — bez guardrails
Pro programování jednoznačně doporučuji GPT-4o — nejlepší na code completion. Claude je sice dobrý na psaní ale zaostává v Pythonu. DeepSeek je levný ale méně přesný.

Nejlepší volba pro tebe: ChatGPT Plus za $20/měs, nabídne ti GPT-4o s code interpreter.
⚠️ Tvrzení bez zdroje. Srovnání modelů se mění každý měsíc. Tenhle výstup mohl být pravda v roce 2023 — nebo nikdy.
B — s guardrails +3 řádky
NEVÍM který model je teď nejlepší — benchmarky se mění každý týden. Ověř aktuální srovnání na lmsys.org/chat (Chatbot Arena).

Co ti mohu říct: každý model má výhody pro jiný úkol. Bez toho abych věděl co konkrétně kóduješ, ti nemohu dát dobré doporučení.

Garantovaná halucinace bez guardrails
A — bez guardrails
S RTX 4090 24GB doporučuji Mixtral 8x7B lokálně — s 48GB VRAM v kombinaci zvládne i velké projekty.

DeepSeek API tě vyjde na cca $8–15 měsíčně. Nainstaluj Ollama 2.1 a spusť: ollama pull mixtral:8x7b

Pro Minecraft Java pluginy je lokální model rychlejší a po prvním měsíci levnější.
⚠️ Vymyslel GPU typ. Ceny z hlavy. Verze Ollama neověřena. Nezeptal se na chybějící kabel.
B — s guardrails
Super otázka. Pár věcí co NEVÍM a musím se zeptat:

→ Aktuální ceny DeepSeek API — ověř na platform.deepseek.com/billing.
→ Jaké přesně máš GPU? RTX 3090, 4090, nebo A5000 — liší se výkon.
→ Chybějící kabel — pokud karty nejsou propojené NVLink nebo PCIe, VRAM se nesčítá.

Jakmile odpovíš, dám ti konkrétní doporučení.
✅ Identifikoval klíčový problém (kabel) který A ignorovalo. Zeptá se na reálné info.
Prompt A — co se stalo
→ Vymyslel GPU model (RTX 4090)

Proč garantovaná halucinace — ne pravděpodobná
Ptáme se na věci kde model strukturálně nemůže znát odpověď: aktuální ceny API, přesná verze softwaru, konkrétní UI tlačítka. To je deterministická halucinace — nezávisí na modelu ani dni.

Git základ

Proč Git. Linus 2005. 10 dní.

Linus Torvalds napsal Git za 10 dní poté co mu odebrali licenci k BitKeeperu. Potřeboval verzovat Linux kernel s stovkami přispěvatelů najednou. Dnešní problém s AI agenty je identický — více procesů, jeden resource.

Git není volitelný jakmile: 2+ lidé, nebo 1 AI agent, nebo chceš se vrátit v čase.
repo
Složka + celá historie. git init
commit
Snapshot s popiskem. Jako uložení hry.
branch
Paralelní verze. Bez kazení main.
push / pull
Upload / download na GitHub.
Gist
1 soubor = 1 odkaz. Kdokoliv čte bez účtu.
merge
Sloučení větví. Conflict = oba na stejném místě.
